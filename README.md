# MMU Student Academic Companion

A modern academic dashboard for Multimedia University students designed to simplify academic management by combining assignments, timetables, results, and notifications into one platform.

---

# Tech Stack

## Frontend
- React
- CSS

## Backend
- Django
- Django REST Framework

---

# Main Features

- Student Dashboard
- Assignment Tracker
- Timetable Management
- Results & GPA Tracking
- Notification Center
- Onboarding Screens
- Mobile Responsive Design

# Installation

## Clone Repository

```bash
git clone <repository-link>
```

---

# Backend Setup

```bash
cd backend

pip install -r requirements.txt

python manage.py runserver
```

---

# Frontend Setup

```bash
cd frontend

npm install

npm start
```
